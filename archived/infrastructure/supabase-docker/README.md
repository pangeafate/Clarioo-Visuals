# Supabase Docker

This is a minimal Docker Compose setup for self-hosting Supabase. Follow the steps [here](https://supabase.com/docs/guides/hosting/docker) to get started.


--

If Chrome refuses to show Basic auth popup, click here: http://supabase:this_password_is_insecure_and_should_be_updated@your-ip-here:8000/. Mind that the page itself won't work properly, you need to head to http://your-ip-here:8000/ after that.
